| lw          | `lw t1, (t0)`    | load content in the address `t0` into `t1` |
| sw          | `sw t1, (t0)`    | store value of `t1` into address `t0`      |

| mul         | `mul a4, a3, a2` | `a4` = `a3` * `a2`                         |
